# Weather App
An iOS application in Swift that aimed to practice the creation of objects such as button, labels and stacks programmatically and 
the make use of JSON data

## Screenshots
![Alt text](/screenshot.png?raw=true "Screenshot") 

## Architecture
MVC 

## Versions
Created on XCode 9 / Swift 4

## Features
+ Displays weather of city inputted from the OpenWeatherMap API
++ description / icon / actual, min and max temperature / wind speed / humidity % / pressure
+ Finds the city name of the current location
